# Content Plan

**Campaign**: "Dein Auto vergisst nichts."
**Duration**: 8-week launch cycle (2 weeks pre-launch + 6 weeks sustain)
**Channels**: Blog/SEO, Twitter/X, Reddit, motor-talk.de, Product Hunt, Hacker News

---

## Week -2 — Seed the Pain (Pre-Launch)

*Goal: Plant the problem in people's minds without mentioning the product.*

## Visual Style Guide (per content type)


| Content Type        | Style                         | Notes                                                                  |
| ------------------- | ----------------------------- | ---------------------------------------------------------------------- |
| App screenshots     | Gradient Cards style          | Use actual app screenshots, phone frame mockup                         |
| Blog headers        | Clean, blue-50 background     | Consistent with app's `from-blue-50 to-indigo-50`                      |
| Social media images | Module gradient as background | Match the module being featured (orange for maintenance, cyan for map) |
| Forum posts         | Text only                     | No images in forum posts — look native, not promotional                |
